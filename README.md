# Oroshi


## Installation (developpement version)

Download the project

```
git clone https://github.com/flaviendelangle/Oroshi.git
cd Oroshi
```

Install the modules for the Django server and the React application
```
pipenv install
cd app
yarn
```

Launch the server
```
yarn start
yarn server
```
